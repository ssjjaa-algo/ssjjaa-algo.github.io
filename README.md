# ssjjaa-algo.github.io

기술 노트 기록을 위한 미니멀 Jekyll 블로그입니다.

## 로컬 개발

```bash
bundle install
bundle exec jekyll serve
```

브라우저에서 `http://127.0.0.1:4000` 접속.
